# Brighton Recruiter

A football analytics platform that identifies players that matches Brighton &amp; Hove Albion's scouting profile

## Goal

Build a scouting dashboard that ranks players using a Brighton Fit Score based on age, position, league, minutes, performance, and estimated market value.

## Target Leagues

- EFL Championship - Local players who has experience in a similar environment (e.g. João Pedro, Georginio Rutter)
- Eredivisie (Kaoru Mitoma, Yankuba Minteh)
- Belgian Pro League (e.g. Moisés Caicedo, Carlos Baleba)
- Argentine Primera División (e.g. Alexis Mac Allister, Facundo Buonanotte)
- Turkish Süper Lig (e.g. Ferdi Kadıoğlu)

## Tech Stack

- Python
- PostgreSQL
- API-Football
- AWS Lambda / S3
- React
- GitHub Actions

## Planned Features

- Collect player statistics from API-Football
- Store cleaned player data in PostgreSQL
- Calculate Brighton Fit Score
- Rank undervalued transfer targets
- Build an interactive dashboard
